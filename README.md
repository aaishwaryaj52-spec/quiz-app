# quiz-app
# 🎯 Quiz App

A simple and interactive **Quiz Application** built using **HTML, CSS, and JavaScript**. This project allows users to answer multiple-choice questions, view their score, and restart the quiz.

---

## 🚀 Features

- ✅ Multiple-choice questions
- ✅ Instant answer validation
- ✅ Score calculation
- ✅ Next question navigation
- ✅ Play Again option
- ✅ Responsive design
- ✅ Beginner-friendly code

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it in your web browser.
4. Start answering the quiz questions.

---

## 🎮 How to Play

1. Read each question carefully.
2. Select one answer.
3. Click **Next** to continue.
4. At the end of the quiz, your score will be displayed.
5. Click **Play Again** to restart.
